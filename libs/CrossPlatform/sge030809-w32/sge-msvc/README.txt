SDL Graphics Extension (SGE)
====================================================================================
http://www.etek.chalmers.se/~e8cal1/sge/index.html
http://home.swipnet.se/cal_home/sge/index.html


This archive contains MS VisualC project files for SGE.

You will need the source code for SGE, SDL (http://www.libsdl.org/) and FreeType2
(http://www.freetype.org). You can also build SGE without support for FreeType2,
see the README in sge/.

These project files are contributed by Seung Chan Lim (limsc@maya.com or 
slim@djslim.com) and are untested by me.
